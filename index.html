Overview

Moyashi URL Shortener APIを使用すると長いURLを扱いやすい短いリンクに変換できます。

HTTPSプロトコルを使用し、JSONフォーマットでデータをやり取りします。

Base Information

Base URL
https://s.moyashi.xyz
HTTPS Required
Content-Type
application/json
Required

URL Shortening Endpoint

POST /shorten
Request Format 
{
  "url": "https://www.example.com"
}
Response Format 
{
  "shortened_url": "https://s.moyashi.xyz/xxxxx"
}

Rate Limits

75
Requests / Minute
Per IP Address
Rate Limit Exceeded
Error
Status Code
HTTP 429
Error Message Too Many Requests
Reset Time
60 seconds

公平な利用とシステムの安定性を確保するため、IPアドレスごとに1分間あたり75リクエストまでに制限しています。

Code Example

Python 
import requests
import json

url = "https://www.example.com"
api_url = "https://s.moyashi.xyz/shorten"
headers = {'Content-Type': 'application/json'}
data = {'url': url}

response = requests.post(api_url, 
    headers=headers, 
    data=json.dumps(data)
)

if response.status_code == 200:
    print(response.json()['shortened_url'])
